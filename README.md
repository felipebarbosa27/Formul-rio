Painel (Dashboard)
Este painel oferece uma visão geral simplificada e objetiva dos dados dos alunos cadastrados no sistema.
Os cards informativos apresentam rapidamente números importantes, como o total de alunos, média de idade, bairro com maior número de cadastros e tipos de responsáveis registrados.

Os gráficos mostram a distribuição de alunos por curso e por bairro, facilitando a análise visual e a identificação de padrões.
Além disso, há listas detalhadas, como:

alunos mais velhos;

alunos mais novos;

CEPs com maior quantidade de registros;

responsáveis com mais de um aluno cadastrado.

O conjunto dessas informações proporciona uma visão completa e estratégica do perfil dos usuários do sistema.

Tela de Registro (Usuários)
Esta tela é responsável pelo cadastro de novos usuários do sistema.
Nela, é possível inserir todas as informações necessárias para criar um acesso, garantindo que cada usuário tenha suas credenciais e dados corretamente armazenados.

Tela de Cadastro (Alunos)
Esta página é destinada ao cadastro de novos alunos.
Aqui é possível inserir todas as informações necessárias sobre cada estudante, permitindo que eles sejam adicionados ao sistema e passem a fazer parte das análises e consultas disponíveis no painel.
